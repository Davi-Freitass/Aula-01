Atividade: Explorando tags basicas

A tag "h" serve para informar um titulo e ele deve ser seguido de um numero de 1 á 6, EX: H1 seria um titulo grande e H2 um pouco menor.

A tag "p" serve para indicar um paragrafo no seu site.

A tag "hr" podemos usar para criar uma linha em determinada parte do seu site.

A tag "br" serve como um "quebra linha"

A tag "a" podemos usar para colocar um link no seu site

A tag "span" podemos usar para dar destaque a uma palavra no seu site como por exemplo: Sesi Senai

A tag "body" é onde o codigo do seu site deve ser escrito.

A tag "lang" é utilizada para definir a linguagem principal do seu site
